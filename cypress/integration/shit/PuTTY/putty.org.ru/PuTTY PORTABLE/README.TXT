
		PUTTY.ORG.RU 0.66-RU-16
		Техническая информация
	~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

В состав пакета PuTTY 0.66-RU-16 входят следующие компоненты:

PuTTY_portable.exe - Основной файл программы PuTTY.

plink.exe     -  Интерфейс командной строки к PuTTY.
pscp.exe      -  Консольный SCP-клиент.
psftp.exe     -  Консольный SFTP-клиент.
pageant.exe   -  Агент SSH-аутентификации для PuTTY, PSCP и Plink.
puttygen.exe  -  Утилита для генерации RSA и DSA ключей.
launcher.bat  -  Менеджер сеансов, альтернатива Pageant.

rz.exe sz.exe -  Файлы пакета lrzsz для работы ZModem.
cthelper.exe  -  Back-end для работы Cygterm.
cygwin1.dll   -  Библиотека для работы компонентов, собранных в среде Cygwin.

putty_zmodem.exe  - Версия PuTTY с поддержкой ZModem.
putty_sizebox.exe - Версия PuTTY с возможностью изменения размера окна настроек.


       CHANGELOG: История изменений PuTTY.ORG.RU:
	~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

0.66-RU-16 - 9 Декарбря 2016

 * Переход русской версии на PuTTY 0.66.
 * При потере соединения PuTTY меняет иконку, сигнализируя о том что
   соединение разорвано.
 * Улучшена работа автоматического переподключения при потере связи.
 * Новый параметр командной строки -auto_store_sshkey для автоматического
   сохранения серверных ключей в программах Plink, PSCP, и PSFTP.
 * Теперь можно вывести список перенаправляемых портов в заголовке окна
   сеанса PuTTY.
 * Новый параметр командной строки -loginscript для загрузки сценария входа
   в систему при старте программы.
 * Появилась возможность создавать скриншоты окна терминала, при нажатии
   «Ctrl+PrintScreen» скриншот будет сохранен в папку с программой.
 * В последовательность Port Knocking теперь можно добавить задержку между
   запросами. Для этого используется специальный протокол «s»,
   например 2001:tcp, 1:s, 2002:udp укажет на паузу в 1 секунду между
   простукиванием портов 2001:tcp и 2002:udp.
 * Пароль теперь может быть пустым для протокола SSH.
 * События мыши теперь не срабатывают, пока окно терминала находится
   в закрепленном режиме («Ctrl+F9»)
 * Новый пункт меню «Экспортировать текущие настройки» для сохранения настроек
   текущего сеанса в файл с расширением .ptx.
 * Новый параметр командной строки -pload для загрузки настроек сеанса из
   файла .ptx.
 * Новый параметр командной строки -fileassoc для ассоциации
   файлов .ptx с PuTTY.
 * Пароль теперь шифруется, находясь в структуре конфигурации в памяти.
 * Новый параметр командной строки -noshortcuts для отключения всех
   горячих клавиш.
 * Новый параметр командной строки -noctrltab для отключения функции «Ctrl+Tab».
 * Интегрирован патч PRINTCLIPPORT.
 * В настройках «Окно/Кодировка» добавлена новая опция «Разрешить рисование
   линий ACS в UTF-8»
 * Полностью переписан патч, отвечающий за создание кликабельных гиперссылок
   («Окно/Гиперссылки»). От старого Hyperlink-патча из проекта NuTTY было
   решено отказаться от из-за большого количества проблем с утечками памяти и
   не совсем корректным регулярным выражением по умолчанию.
 * В Pageant интегрирован патч, добавляющий подтверждение ключей при их
   использовании, в стиле OpenSSH.
 * Исправлен сбой в ZMODEM-версии PuTTY при использовании SSH туннелей.
 * Переключение окон сочетанием клавиш «Ctrl+Tab» (опция доступна в настройках
   «Окно/Поведение» для не запущенного сеанса) не работало при только лишь
   двух окнах.
 * Исправлен сбой программы при вводе очень длинного пароля.
 * Исправлена ошибка, при которой в портативном режиме не было возможности
   сохранить текущий сеанс под другим именем.
 * Исправлена утечка памяти при использовании фоновых изображений.
 * При изменении размера окна, обновление размера происходило на протяжении
   всего процесса изменения. Теперь это делается только по завершении.
 * Исправлен сбой программы при быстром двойном клике по кнопке «Соединиться».
 * Опция «Окно имеет кнопку Закрыть» в настройках «Окно/Поведение/Меню»
   не работала при старте программы, а только при реконфигурации запущенного
   сеанса.
 * Исправлена утечка памяти в функции auto-password
 * В управлении сеансами больше нельзя создать папку с именем Default.
 * Исправлена работа опции «Отправка пустых пакетов для сохранения активности
   сеанса» в настройках «Соединение».
 * Опция «Выделять полужирный текст изменением» в настройках
   «Окно/Цветовая схема» не сохранялась.
 * Исправлена утечка памяти при закрытии окна конфигурации запущенного сеанса.
 * Параметр командной строки -pass больше не работает.
 * Исправлены ошибки при сохранении границ окон в настройках «Окно/Внешний вид».
 * Другие утечки памяти, незначительные багфиксы, и исправления локализации.

0.63-RU-15 - 15 Мая 2014
 * Поддержка технологии Port Knocking (протестировано с knockd).
 * В системном меню появилось новое информационное окно
   «Перенаправления портов».
 * Интегрированы некоторые функции из патча TuTTY
 * Добавлена возможность спрятать кнопки системного меню «Свернуть окно»,
   «Развернуть окно», «Закрыть окно» (в настройках Окно/Поведение/Меню).
 * В настройках цветовой схемы добавлена возможность выбрать
   альтернативный шрифт для подчёркнутых и выделенных символов.
 * Параметр задержки до автоматического переподключения теперь настраивается.
 * Добавлено новое поле комментария к сеансу (в настройках Сеанс/Комментарий).
 * Новый механизм функции сценария входа в систему: содержимое файла теперь
   сохраняется вместе с сеансом, исходный файл может быть удалён.
 * Аутентификация с ключом работала некорректно, если не был запущен Pageant.
 * Сочетания клавиш «Ctrl+Left» и «Ctrl+Right» теперь работают.
 * Прокрутка мышью с огромным значением «Обратная прокрутка» приводила к сбою.
 * Окно в панели задач не мигало при получении кода BELL.

0.63-RU-14 - 12 Января 2014
 * Новый патч PuTTYwincrypt для работы со Smart Card и хранилищем
   сертификатов Windows.
 * Переключение между окнами при помощи «Ctrl+Tab» (Окно/Поведение).
 * Рабочая директория PuTTY, директория конфигурации, и содержание
   переменной PUTTY_PATH теперь добавляются к переменной окружения PATH
   (используется для поиска стороннего ПО, например winscp.exe).
 * Переменная окружения PUTTY_INI_FILE служит для задания пути к файлу
   конфигурации putty.ini.
 * Увеличение производительности в PuTTY.exe.
 * Увеличение производительности в PSFTP.exe.
 * Заголовок окна больше не должен сбрасываться при восстановлении PuTTY из
   панели задач.
 * Патч Hyperlink переписан с C++ на C.

0.63-RU-12 - 11 Ноября 2013
 * Переход русской версии на PuTTY 0.63.
 * Добавлена совместимость с 64-битной версией cygterm.
 * Новая опция «На полный экран при запуске» и «Развернуть при запуске» (в
   настройках Окно/Поведение).
 * Добавлена опция «MouseShortcutsFlag» для отключения сочетания клавиш мыши.
 * Добавлена опция «AutoStoreSSHKeyFlag» (а так-же флаг -auto_store_sshkey) для
   авто-сохранения серверных SSH ключей.
 * Внедрён патч «KEYMAPPINGPORT», исправляющий сочетания клавиш для
   работы с Emacs и Vim.
 * Полностью переписана функция автоматическое выполнения команд.
 * Улучшена работа фильтра сеансов в портативном режиме (фильтр работал очень
   медленно при большом количестве сохранённых сеансов).
 * Исправлена работа флага -cygterm.
 * Обновлена библиотека регулярных выражений, исправлены утечки памяти.
 * Исправлены проблемы с патчем Zmodem.
 * Исправлена двойная отправка login при автоматическом telnet-соединении.
 * Исправлено определения гиперссылок при пустом регулярном выражении.
 * Сеансы с сохранённым паролем теперь можно дублировать.
 * Исправлена утечка памяти в функции управления заголовками окон.
 * Функция auto-command могла не работать в портативном режиме.
 * В некоторых случаях сохранение пароля приводило к зависанию PuTTY.
 * Фильтр сеансов по папкам не работал, если имя папки содержало слэш.
 * Размер logon-скрипта был статический (размер буфера 4096),
   теперь он динамический.
 * Незначительные багфиксы и исправления локализации.

0.62-RU-11 - 23 Марта 2013
 * Добавлена поддержка протокола IPv6 при интеграции с PSCP и WinSCP.
 * Улучшена функция открытия соединения в новом окне.
 * Исправлена ошибка с нестандартным DPI.
 * Исправлено повреждение файла putty.ini в портативном режиме.

 * Убраны конфликтные горячие клавиши (печать по F7 и SHIFT+F7).

0.62-RU-10 - 21 Января 2013
 * Добавлен таймер ротации log-файлов в настройках журналирования PuTTY.
 * Добавлена кнопка открытия соединения в новом окне (панель настроек при этом
   не закрывается).
 * Новая кнопка «Очистить» для сброса поля «Управление Сеансами» в окне
   конфигурации PuTTY.
 * Если сеанс SSH разорван, горячая клавиша «CTRL+SHIFT+left» теперь
   используется для перезапуска.
 * Добавлено сохранение имени пользователя при вводе с клавиатуры по протоколу
   SSH.
 * Новая опция в настройках уведомлений терминала, при сигнале переводящая окно
   PuTTY на передний план.
 * Список сохранённых сеансов теперь обновляется, если сеанс был удалён.
 * Новые макро-шаблоны для добавления метки времени log-файлов: %f для
   миллисекунд и %t для табуляции.
 * Исправлена ошибка в портативном режиме PuTTY в фильтрах сеанса.
 * Новая опция «WinSCPProtocol» для выбора протокола для WinSCP (scp/ftp/sftp).
 * Новая опция «UserPassSSHNoSave», чтобы избежать сохранения имени пользователя
   и пароля во время SSH-аутентификации.
 * Новое контекстное меню в менеджере сеансов Launcher.
 * Новый пункт меню «Обновить» в менеджере сеансов Launcher.
 * В менеджере Launcher нет больше сеанса по умолчанию «Default Settings».
 * Незначительные баг-фиксы и исправления локализации.
 * Поддержка ZModem отключена по умолчанию, т.к. патч являлся поводом для
   проблем при использовании X11 переадресации. Для работы по протоколу ZModem
   используйте putty_zmodem.exe.
 * Из-за проблем, вызванных переключением между кодировками в терминале cmd.exe,
   было принято решение отказаться от локализации интерфейса командной строки в
   программах PLINK, PSCP, PSFTP.
 * Версия PuTTY с возможностью изменения размера окна настроек теперь носит имя
   putty_sizebox.exe.

0.62-RU-9 - 20 Февраля 2012
 * PuTTY больше не сохраняет пароли в памяти по ошибке.
 * Pageant теперь работает как с новым стилем настроек (PuTTY 0.61 и выше),
   так и со старым (0.60 и ниже).
 * PuTTY больше не печатает ложные «Access denied» при неудачной
   аутентификации GSSAPI.
 * PSCP и PSFTP теперь понимают нестандартные номера портов в сохраненных
   сеансах SSH.
 * В Pageant исправлена утечка дескриптора файла при неудачной аутентификации.
 * Исправлено падение PuTTYtel при сохранении сеанса.
 * PuTTY теперь рисует нижнее подчёркивание под подчёркнутым текстом, вместо
   переноса её куда-то вправо.
 * PuTTY теперь не должен печатать символы рисования линий VT100 при
   неправильном вертикальном смещении.
